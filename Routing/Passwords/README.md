# Password Security Labs

This folder contains labs related to router password security in Cisco Packet Tracer.

## Labs Included
1. **Enable Password Encryption Toggle**  
   - Explore enabling and disabling password encryption on routers.  

2. **Enable Password vs Enable Secret**  
   - Compare `enable password` and `enable secret`, and observe which is encrypted.  

3. **Console Password and Encryption**  
   - Configure console access password, test encryption, and secure the router.  

---

Each `.pkt` file includes full instructions inside the Packet Tracer simulation.  
Open the files in Cisco Packet Tracer to view step-by-step objectives and configurations.
