enable
configure terminal

! Create VLAN 2
vlan 2
 name VLAN2
exit

! Assign PC3 port to VLAN 2
interface fa0/2
 switchport mode access
 switchport access vlan 2
exit

! Configure trunk link to SW1
interface fa0/1
 switchport trunk encapsulation dot1q
 switchport mode trunk
 switchport trunk allowed vlan all
exit
